org.apache.jasper34.utils
Interface Expirable

public interface Expirable

Interface for objects that have an expiration. Used by the SimpleCache class to expire objects stored in it. Objects that wish to be expired when stored in the cache should implement this interface. It is not necessary to implement this interface to store objects in a SimpleCache.
