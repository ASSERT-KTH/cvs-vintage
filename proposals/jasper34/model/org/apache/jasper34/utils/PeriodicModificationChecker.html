public class PeriodicModificationChecker
extends SimpleModificationChecker

implementation of Modification Checker that will only perform a check periodically, otherwise returning the last status check result. Adds a setModified() method for the client to assert the status mid-period (i.e, right after a successful recompile, to assert modified==false). The asserted status only lasts until the current status check period.

REFRESH_PERIOD_PROPERTY

public final static String REFRESH_PERIOD_PROPERTY
name of property to set to control how often the JSP compiler will check modification dates on the Jsp files. This is the minimum period (in seconds) before a modification check will take place. For improved performance, set this to a large value. 

org.apache.jasper.mod_check.refresh_period = 120
The default value is 60 seconds.
