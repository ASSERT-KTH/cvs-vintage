public class JspRecursiveModificationChecker
extends SimpleModificationChecker

implementation of Modification Checker that will recurse through all static includes in a Java Server Pages source document (up to a depth of INCLUDE_RECURSION_MAX), comparing file modification dates until it finds one newer than the target file. Otherwise, returns false.
