Field Detail

JSP_CACHE_SIZE_PARAM

public final static String JSP_CACHE_SIZE_PARAM
name of init-param to set for the JspServlet to indicate the size of the JSP page handler cache. This number should correlate with the number of JSP files hosted by the server. Example: 
<init-param>
<param-name>cache_size</param-name>
<param-value>128</param-value>
</init-param>
If not set, the default is to use the value set for org.apache.jasper34.utils.CacheDefaults.JSP_CACHE_SIZE

JSP_CLASSPATH

public static String JSP_CLASSPATH
classpath used to compile and run JSP servlets.

JSP_CLASSPATH_PROPERTY

public final static String JSP_CLASSPATH_PROPERTY
name of property or init-param to set to tell JSP compiler which classpath to use when compiling/running JSP pages. 

org.apache.jasper.jsp_classpath = ....
If this is not set, will default to use value set for the property "org.apache.tomcat.jsp_classpath". If neither is set, will fallback to default class loader's classpath.

JSP_PROTECTION_DOMAIN

public final static String JSP_PROTECTION_DOMAIN
ProtectionDomain attribute for JSPs. This is tomcat specific. Other servlets engines can choose to have this attribute if they want to have this JSP engine running on them.

JSP_SERVLET_BASE

public final static String JSP_SERVLET_BASE
The default base class of the generated servlets.

JSP_SERVLET_BASE_PROPERTY

public final static String JSP_SERVLET_BASE_PROPERTY
Property for specifying the base class of the generated servlets. 
org.apache.jasper.jsp_servlet_base =
This sets the default base class extended by servlets generated by the Jsp compiler. This can be overridden on each page by using the '<%@page extends = ....%>' tag. The default value is "org.apache.jasper.runtime.HttpJspBase".
