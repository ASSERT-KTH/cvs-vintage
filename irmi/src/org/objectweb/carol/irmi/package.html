The irmi package is a JDK agnostic RMI implementation supporting pluggable interceptors and local call optimization. It uses standard JRMP stubs and uses the {@link javax.rmi.CORBA.PortableRemoteObjectDelegate} interface to plug into any JDK. This implementation may be enabled by setting the system property named javax.rmi.CORBA.PortableRemoteObjectClass to the value "org.objectweb.carol.irmi.PRO".
