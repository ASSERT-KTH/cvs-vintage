Scarab - Issues

As of 7-9-2002, Scarab is now self hosting!

Scarab is now used to track Scarab issues. Previously, we were using the Tigris.org IssueZilla issue tracker. Please note that we have not finished the migration from the old system to the new system quite yet, so please be patient.

If you would like to submit an issue, please search the old issue tracker first to see if the problem has already been reported. If you cannot find a problem, then submit it to our new issue tracker (ie: the Scarab one).

Also, please make sure to check the KNOWN_ISSUES.txt file to make sure that we don't already know about an issue that may not necessarily have a particular issue the problem is reported against.

If the new issue tracker is unavailable or you have a problem with it, please contact Jon and let him know.
