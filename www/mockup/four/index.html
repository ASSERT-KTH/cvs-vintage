Terms

 Issue A bug or feature request, the type is one of an enumeration defined in a database table
 Platform  The type of hardware involved
 OS  The operating system involved
 Project  The development project
 Component  The part of the software where the bug is located or where the new feature should be added
 reporter  person who initially enters the issue report
assigned developer person who should fix the bug or implement the new feature
QA contact person who should test that the patch actually resolves the problem
milestone abstract time by which the issue shoud be resolved, usually corresponds to a planned release
Severity how bad was this bug?
Priority how important is it to fix this bug?
Status has this bug been assigned to a developer, resolved, rejected, defered, etc.?
Resolution If the issue has been resolved, how was it resolved: fixed, rejected, defered, by-design, etc.?
Vote Each developer has a set of votes that can be applied to one or more issues to emphsize them
activity a log of all changes made to all bugs
keyword a defined list of keywords that help to briefly describe an issue
attachment a data file accompanying an issue report, analygous to an email attachment
version a numbered release of the product
dependcy when one bug cannot be resolved until another has been resolved, it is said to depend on it
subscriptions users who are interested in a bug or set of bugs can subscribe to notifications of any changes to those bugs


Classes of Users

Class of user Abbrev. Description
 End User EU  Someone who simply uses the product being developed
 Developer D  A person working on the development of the product
 Core developer CD  A person who has shown significant dedication to doing development on the project
 Project Leader PL  The person(s) who founded the project and have the most ability to control it
 Bug Czar BC  The person(s) who take responsibility for organizing/maintaining the issue database
 Web Administrator WA  Person maintaining the host machine where the scarab system is intalled and running
