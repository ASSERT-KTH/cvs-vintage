Introduction

When building the dependencies for Scarab, there are several things to consider. For example, some of the dependencies have a requirement to set the Xerces parser to be the 1.4.x version instead of the 2.x version.

Xerces notes

Projects which require to set the Xercies 1.4.x version (by creating a $PROJECT/build.properties (xerces.jar=/path/to/xerces-1.4.4.jar):

  • jakarta-turbine-3
  • jakarta-turbine-fulcrum
  • jakarta-turbine-torque

Fulcrum notes

The jakarta-turbine-fulcrum project MUST be built using the Scarab build.xml: upgrade-fulcrum target. This is because a couple of build time settings must be made when building fulcrum so that it can work with the Scarab database. In order to use this target, the jakarta-turbine-fulrcum project must be checked out in the same directory where the scarab directory lives. It is suggested that you first make sure that fulcrum will build on its own and then you can try using Scarab's build system to build/upgrade fulcrum in Scarab.
