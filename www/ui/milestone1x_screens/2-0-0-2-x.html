Defect: Scarab functions

General user functions

   

Administrative functions

Enter artifact
Use this form to define a new instance of an artifact - Each artifact type (e.g. defect, enhancement, etc.) has it's own corresponding form for generating new entries. If you are entering multiple artifacts of the same type, and find yourself repeatedly entering the same fields over and over (example: platform-Sun, OS-Solaris, Functional area-documentation) you may choose to save a partially completed "Enter artifact" form as a template. Thus, selecting this template would start you off on an "Enter artifact" form pre-populated with your selected entries (the interface for template selection appears at the top of the "Enter artifact" form.

Artifact template list
List of all "Artifact templates" available to role. User can access template details, in order to view or edit, by clicking on the templates name. User selects to use a template via a pop-up menu at the top of the enter isse page. This template remains as the default until another selection is made.

Saved query list
List of all saved queries avaible to role. User can access saved query details , in order to view or edit, by clicking on the query's name, and execute a query by clicking on it's associated project. Saved queries are also available through a pop-up menu at the top of every screen within scarab.

Generate report
Use this wizard to create reports on Scarab module activity.

   
Approve / reject user submissions
Review and respond interface for user submissions including, role promotions, saved queries, templates, automated alert emails, etc.
Manage "Artifact types" form
Add and sort Artifact types within a module. The interface on this screen includes options to activate/inactive Artifacts, along with choosing five artifact that will appear in the left hand navigation of Scarab.
Manage "Enter artifact" form
Add, edit, create, group, and arrange attributes on the "Enter artifact" form.
Global attributes
Master list of all attributes, both out of the box, and custom, available for use when creating/editing the enter "issue form".
