General user functions

Administrative functions

Enter issue template list
List of all enter issue templates available to role. User can access template details, in order to view or edit, by clicking on the templates name. User selects to use a template via a pop-up menu at the top of the enter isse page. This template remains as the default until another selection is made.

Saved query list
List of all saved queries avaible to role. User can access saved query details , in order to view or edit, by clicking on the query's name, and execute a query by clicking on it's associated project. Saved queries are also available through a pop-up menu at the top of every screen within scarab.

Vote management
This list displays a user's current open votes, and a vote management interface they can use to rescind/reprioritize votes, or to append comments to votes.

Auto alert email options
Subscription/frequency options for auto alert emails available per project.

Approve / reject user submissions
Review and respond interface for user submissions including, role promotions, saved queries, templates, automated alert emails, etc.

Manage "enter issue" form
Add, edit, create, group, and arrange attributes on the "enter issue form".

Global attributes
Master list of all attributes, both out of the box, and custom, available for use when creating/editing the enter "issue form".

Vote management
Project level settings for votes. Admin settings include determining when a vote is added to an issue (manual or automatic), how many votes are required to force a state change, etc.

Auto alert email list
Manage, edit, and create auto alert emails used for automated workflow management within a Scarab project.
