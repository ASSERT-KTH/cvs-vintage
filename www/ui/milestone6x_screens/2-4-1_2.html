Define report (2.4.1): Defining a base set of issues

This selection limits the set of issues that this report will be drawn from.

This selection is useful (but not required) if you want to limit your report to a particular set of attribute(s)/option(s) (e.g. "selecting the attribute state" or "status", limiting the report to options such as "open", "open unassigned", "open assigned", "closed", or a combination of several of these options).

Step two of Defining a base set of issues allows you to group together attributes and/or options that you wish to appear as column headings in the final report (e.g. X axis).

note: Selecting the attribute's name will select all options within that attribute.
