Select primary filter attribute(s)

Primary filter attribute
This selection defines the subset of issues that this report will be drawn from. It is often useful (but not required) to set the primary filter attribute "state" or "status", limiting the report to options such as "open", "open unassigned", "open assigned", "closed", or a combination of several of these options.

note: Selecting the attribute's name will select all options within that attribute. If you choose to "continue", without defining PFA groups, each selected Attribute and/or Attribute Option will appear as a seperate column (i.e. side-by-side comparison).
