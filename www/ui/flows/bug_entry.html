Viewing and Entering a Issue


Viewing and Entering a Issue:

1. User wants to view bugs
   --> If Logged in: StartPage
   --> If not logged in: TLogin

2. User reviews their StartPage selects Project "manhattanproject"
   --> Logged in: ProjectView
   --> Not logged in: TLogin

3. User wants to enter a bug, clicks "Enter New Issue"
   --> Logged in: NewIssueStep1
   --> Not Logged in: TLogin

3a. User enters data into NewIsuseStep1 and clicks "Next Step"
   + Scarab searches for possible matches
   --> If matches: IssueMatches
   --> If no matches: NewIssueStep2
   
3b. User enters data into NewIssueStep2 and clicks "Next Step"
   --> NewIssueStep2

3c. Scarab displays IssueDetail
   --> IssueDetail
