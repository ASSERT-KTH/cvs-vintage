Project Scarab Administration

If you have administrative privleges on this project, you can login here.

The administrative pages allow project leaders to

  • grant cvs commit privleges,
  • create or modify CVS modules,
  • create or modify mailing lists,
  • configure other hosted tools as they become available.

To get involved, join the developer's mailing list by sending email to dev-subscribe at scarab.tigris.org.
