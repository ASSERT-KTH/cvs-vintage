Development Tools and Technologies...

Here is a list of technologies that are currently slated to be used within the project.

  • Turbine will be used as the underlying framework
  • Peer model and Torque within Turbine will provide the database abstraction
  • Velocity will be used as the abstraction between the user interface and the business logic. The look and feel of this application easily customizable.
  • MySQL will be the SQL backend (Support for other RDBM's will also be included. We are just developing with MySQL for now because it is a commonly used database.)
  • Tomcat will be the target Java Servlet engine (Any Servlet API 2.1 or higher engine will work).
