Team | File Content
Make sure the following file extensions are present, and are labelled ASCII.
  • bat - MS Windows batch file
  • cmd - MS Windows command file
  • css - Cascading Style Sheet
  • dtd - DTD
  • js - JavaScript
  • pl - Perl script
  • sh - Bourne Shell script
  • sql - SQL script
  • vm - Velocity macros
  • vsl - Velocity macros
  • wsdd - Web Service Deployment Descriptor
  • xsl - XML style sheet
