This documents the process for doing a release of Scarab. It has been deprecated and you should now refer to How to Release.

  1. Make sure your working copy has no local modifications and is up-to-date. (Starting with a fresh copy is preferred.)
  2. Build and execute tests.

    cd scarab/build
    ant clean tests

  3. Fix any unexpected failures and commit changes.
  4. Check out a fresh copy of Scarab.

    cvs co scarab
  5. Edit scarab/build/default.properties and change the version number from the -dev to the final version number for the release.
  6. Update the scarab/www/index.html file to announce the new release as well as update the scarab/www/changelog.html file.
  7. Check both the default.properties and index.html files back into CVS.
  8. Tag cvs with the release tag. Mail the dev@scarab mailing list to let people know that they should hold off on commits until the message comes through that you are done with the tag/release process. Change the version number in the example below to the version of the current release.

    cd scarab
    cvs tag SCARAB_1_0_B17
  9. Build a release. This will create a .tar.gz and a .zip file in the scarab directory.

    cd scarab/build
    ant package
  10. Edit scarab/build/default.properties and change the version number from the final version number to the -dev version and check this file back into CVS.
  11. Log into the Scarab website with your account. Go to the Downloads area (http://scarab.tigris.org/servlets/ProjectDocumentList) and move the existing released files into the "Old releases" folder.
  12. Upload the new releases into the "Top" folder.
  13. Announce to the announce@scarab.tigris.org, users@scarab.tigris.org, dev@scarab.tigris.org lists that there is a new release available: 
        Subject: [ANNOUNCEMENT] Scarab 1.0 beta 17 released

    The Scarab team is pleased to announce the release of version 1.0
    beta 17 of the Scarab issue tracking system.

    This is primarily bug fix release.

        http://scarab.tigris.org/
  14. Post a new release on the freshmeat.net Scarab entry.
