Scarab Project Mailing Lists

To subscribe or unsubscribe, send an email message to the email address associated with each "subscribe" or "unsubscribe" line below. The subject line and message body are not used, so you may leave them blank.

You must subscribe before you can post to any of these lists. This rule helps to prevent spam.

announce@scarab: Announcements of new releases and other major project events.

Subscribe: announce-subscribe@scarab.tigris.org
Unsubscribe: announce-unsubscribe@scarab.tigris.org
 
dev@scarab: The main list for developers to discuss development issues, and general discussion of bugs and bug fixing.

Unsubscribe: dev-unsubscribe@scarab.tigris.org
 
bugs@scarab: Automated notifications of bugs reported against scarab itself.

cvs@scarab: Automated notifications of source code changes.
