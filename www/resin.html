I have Scarab successfully running under the Resin (http://www.caucho.com)
application server.  Jon Stevens suggested I post what is necessary to get
Scarab running with Resin.

The good news is that there are no special steps.  I just built the sandbox
and then copied the resulting web application to the Resin webapps
directory.  That is:

$ cp -R scarab-VERSION/target/webapps/scarab /usr/local/resin/webapps

I then restarted Resin.  The URL to access the scarab install on Resin is:

http://server.domain.com:port/scarab/issues

Since the scarab webapp is not configured to be the root webapp.
