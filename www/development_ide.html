Using an IDE to Develop Scarab

There is a lot of value in having the Scarab CVS distribution be IDE-ready. Facilitating easy development of Scarab with a GUI IDE should go a long way towards helping new developers get up to speed.

Scarab CVS is configured to be IDE-ready for the following GUI IDEs:

  • Eclipse (Open Source)
  • Websphere Studio (Commercial, IBM, based upon Eclipse)

The following guidelines should assist with any developer who wants to add support for a favored IDE.

  • Changes to support development using an IDE must be reasonably transparent to the build process. At a minimum, Scarab will be buildable without any additional configuration by the existing build process, which is documented in build/nightly.sh and uses Apache Ant.
