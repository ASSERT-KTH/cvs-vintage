Progress

Overall:

  • IA/UI has been started.
  • Integration issues have been started and nearly completed.
  • Module code has been abstracted out using a Factory and Services pattern so that it is possible to integrate Scarab into other systems.
  • Implementation of Screens has been started (with many completed).

Specific:

  • Login/Registration has been completed.
  • Enter an issue has been completed.
  • View an issue has been completed.
  • Query for an existing issue is almost completed.
  • Attribute editor is almost completed.
  • Integration of Flux has been completed.
