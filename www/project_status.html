Details

The Scarab project is currently under development. It has been under development for quite some time now, however the development hasn't been 100% on Scarab, it has been on the underlying framework technologies that we are using to develop Scarab. The reason is that we have wanted to solve some of the MVC design issues (Pull Model and Intake) before trying to create the application. This development has gone on in the Turbine and Velocity projects on jakarta.apache.org
