Tentative Roadmap

Scarab 1.0dr1* Release Date: 10/15

* "dr" stands for Developer Release which means that it is still under heavy development and is not considered ready for use on live servers. We expect to have weekly dr releases until we get to either alpha or beta releases.

Project Milestones, who is assigned to what and their dates of completion are now tracked on another page. Note: the finish date says 12/3. Thanks to the magic of M$ Project, this isn't accurate. We are shooting for a decently usable system around the middle of October.

There is a Runbox where you can look at an actual instance of Scarab running on a machine. This is a nightly build and run of the latest version of Scarab from CVS. Any data in the database will get replaced the next rebuild. Note: If it shows an error is not responding, it is probably due to a bad nightly build and will be corrected within the next 24 hours.
