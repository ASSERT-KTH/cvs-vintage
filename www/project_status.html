Recent Activities

Because things change quickly and we may not have time to always update this document, it is best to consider the mailing list archives the most up to date reflection of the current project status.

  • 12-6-01 - Thanks to the excellent work of the guys at Bitonic we now have the ability to transform a Bugzilla XML file into a Scarab XML file and can then import that into the Scarab database. This is a huge milestone for us. We have all been working on tidying up lose ends and making sure that the flow of the application is appropriate. We are also focusing on making the UI easier to use in areas that we have already identified. Things are still progressing very well.
