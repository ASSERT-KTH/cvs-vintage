Screen Description: IssueList

Name: IssueList
Purpose: List results of running search. Ability to sort results and customize columns.
Contents:
  1. Table of search results
    • columns sortable (issuezilla can currently sort all the clicked columns, but it cannot sort all of the clicked columns and then change the sort order (ie: top/bottom->bottom/top) of a selected column. that would be a nice feature. example: sort by date + priority and have p1 sort before p2 or be able to reverse it to have p2 sort before p1)
    • each issue a link to IssueView
  2. Link to configure columns
  3. Link to long results mode
  4. Link to modify multiple mode
