Screen Description: Attributes

Name: Attributes
Purpose: Allows the Domain Admin or Project Owner to add, edit and/or delete project attributes (fields)
Contents:
  1. Basically a screen that will display all existing attributes (i.e. Status, Priority, Component, etc). and attribute options (i.e. Open, 1, NewProject, etc)
  2. User should be able to edit existing attributes and attribute options, delete and /or add new attributes and attribute options
  3. Submit and Cancel buttons
