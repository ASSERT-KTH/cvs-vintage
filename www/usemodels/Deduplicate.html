Screen Description: Deduplicate

Name: Deduplicate
Purpose: Show user a list of existing issues that might match the one the user thought to enter, thus reducing duplicate issues.
Contents:
  1. List of potential matching issues
  2. ability to indicate which existing issues (if any) are the same as the new one
  3. ability to indicate that no existing issues are the same
  4. ability to add a comment to an issue.
  5. ability to add a vote to an issue.
