Screen Description: DefineIssueStatus

Name: DefineIssueStatus
Purpose: Allows the Domain Admin or Project Owner to customize both the Issue Status and the Status Transition Model Workflow.
Contents:
  1. Basically a screen that will display all existing statii (i.e. Open, Closed, etc) and the existing transition workflow (i.e. New-Start-Assigned-Fixed-Closed)
  2. User should be able to edit existing statii and transition workflows, delete and /or add new statii and transition workflows
  3. Submit and Cancel buttons
