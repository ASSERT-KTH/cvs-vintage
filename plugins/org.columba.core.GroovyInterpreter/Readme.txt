1. Introduction

After installing this plugin one is able to write plugins in the Python programming
language (http://www.python.org). This is possible using Jython (http://www.jython.org),
a Java implementation of Python.

2. Installation
Select Utilities->Plugin Manager to open the Plugin Manager Dialog. Select the
"Install..." button which opens a dialog where you have to choose the location
of the zip-file containing your plugin.
