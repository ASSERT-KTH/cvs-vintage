Directory structure

The files contained in this distribution are contained in the following directories.

  1. docs This documentation
  2. lib Contains all libraries that are necessary in order to build jBoss
  3. src/* All sources needed to build the distribution of jBoss are contained here
  4. src/bin Contains the scripts used to start jBoss and additional tools
  5. src/build Contains the build scripts. Use these to build jBoss
  6. src/docs Contains documentation for the jBoss server
  7. src/etc Contains various configuration files such as manifest files and KAWA project file. Also contains the code template used by jBoss developers.
  8. src/etc/beaninfo Contains all EJX plugin BeanInfo information files. These are used to construct BeanInfo classes by the build scripts, which are used in the EJX GUI to enter deployment information about EJB's
  9. src/etc/conf Contains the default configuration files for jBoss
  10. src/lib Contains all libraries that are necessary in order to run jBoss. These are copied into the /dist/lib/ext directory by the build scripts
  11. src/main Contains the Java source files for jBoss
  12. src/resources Contains all resource files that are used by jBoss classes. Resource files should be placed in the same package as the class that is using it. These will be merged into the distribution by the build scripts so that they are available on classpath.
